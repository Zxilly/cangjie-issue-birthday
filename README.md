# 蛋糕日 · CakeDay

为 [Cangjie](https://gitcode.com/Cangjie) 组织中长期未关闭的 issue 送上生日祝福的小工具。

使用 GitCode 账号登录后，列出当前用户在 Cangjie 组织创建、仍处于 open 状态且已满 1 周岁的 issue，标注其周岁数，并可在对应 issue 下发表一条祝福评论。未满 1 周岁的 issue 也会一并列出，并显示距下一个周年的天数。

线上地址：<https://birthday.cj.zxilly.dev>

## 技术栈

React 19、TypeScript、Vite、Tailwind CSS v4，部署于 Cloudflare Pages。

开发、构建与部署说明见 [DEVELOPE.md](./DEVELOPE.md)。
